# EchoLine — 老友记英语学习播放器

移动端优先的网页播放器：用户端看剧集列表、点开即学；管理端一集一集上传、一条一条调轴，保存后即时发布。

## 运行方式

1. 安装 Node.js（若尚未安装）。
2. 在项目目录执行：
   ```bash
   node server.js
   ```
3. 浏览器打开：
   - **用户端**：http://localhost:3000/
   - **管理端**：http://localhost:3000/admin.html

## 管理端（开发阶段）

- **添加一集**：填写标题，选择视频文件、英文字幕 SRT，可选中文字幕 SRT，点击「上传并发布」。该集会立即出现在用户端列表。
- **逐条调轴**：在已上架剧集中点击某集的「调轴」，在编辑器中可修改每条字幕的开始/结束时间（输入框或「设为开始」「设为结束」从当前视频位置取时），保存后用户端将使用新字幕。

## 用户端

- 首页展示已上架的完整剧集列表，点击某一集进入播放页。
- 播放页：上半屏视频，下半屏字幕；点击某句字幕跳转并播放；「上一句」「下一句」快捷导航；语速 0.5x～2.0x；当前句高亮。

## 项目结构

- `index.html` — 用户端入口（剧集列表 + 播放页）
- `admin.html` — 管理端（上传、调轴）
- `css/style.css` — 淡雅简洁样式
- `js/app.js` — 剧集列表与播放页逻辑
- `js/player.js` — 视频与字幕渲染、语速
- `js/subtitles.js` — SRT 解析与双轨
- `js/subtitle-sync.js` — 点击跳转、高亮、上一句/下一句
- `server.js` — 本地静态服务 + 上传/更新 API
- `data/episodes.json` — 剧集列表（由管理端更新）
- `videos/`、`subtitles/` — 视频与字幕文件（由管理端上传生成）
